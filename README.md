# 🛒 SmartCart: Ideas de Recetas para Mamá

SmartCart es una aplicación de planificación de comidas diseñada para simplificar la organización semanal. Permite gestionar un catálogo de recetas, planificarlas en un calendario interactivo y generar automáticamente listas de la compra.

## 🚀 Versión Serverless (Actual)

Esta aplicación es ahora **100% Frontend**. Funciona íntegramente en el navegador:
- **Sin Backend**: No necesitas configurar Node.js ni bases de datos externas.
- **LocalStorage**: Tus planes y recetas personalizadas se guardan en tu propio dispositivo.
- **Listo para Netlify**: Despliegue instantáneo con un solo click.

## 🛠️ Instalación y Despliegue

### Local
```bash
cd frontend
npm install
npm run dev
```

### Despliegue en Netlify
1. Ejecuta `npm run build` en la carpeta `frontend`.
2. Sube la carpeta `dist` resultante a Netlify.
3. ¡Pásale el enlace a quien quieras!
